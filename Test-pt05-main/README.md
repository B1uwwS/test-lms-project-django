команды для активации .venv - виртуальное окружение 
на windows - .venv\Scripts\activate
на linux/macOS/ubuntu - source .venv/bina/activate 

pip install django
django-admin startproject core
ls - проверка директории, где я нахожусь 
pwd - показывает весь маршрут пути
cd [название папки] - возможность сменить директорию
python manage.py runserver - запуск сервера
python manage.py startapp blog - создание приложений 


python manage.py makemigrations - подготовка к миграции 
python manage.py migrate - проводим миграции
python manage.py createsuperuser - создание супер - пользователя
